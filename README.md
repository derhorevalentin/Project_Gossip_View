# README

Première partie pour un futur site Gossip qui déchire.

Instruction à suivre:

* `1 --> bundle install`

* `2 --> rails db:create`

* `3 --> rails db:migrate`

* `4 --> rails db:migrate:status`

* `5 --> rails db:seed`

* `6 --> rails server`

* `7 --> Go le navigateur et tape: localhost:3000`

* `8 --> Enjoy !! ;)`
